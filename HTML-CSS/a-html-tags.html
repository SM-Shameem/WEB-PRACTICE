04 title # global display: none all

# defines the title of the document, the title must be text-only, and it is shown in the browser's title bar

# is required in html documents

# the content of a page title is very important for search engine optimization (SEO)

# the page title is used by search engine algorithms to decide the order when listing pages in search results

# you can not have more than one (title) in an HTML document

# the title -

> defines a title in the browser toolbar

> provides a title for the page when it is added to favorites

> displays a title for the page in search-engine result

# here are some tips for creating good titles -

> go for a longer, descriptive title (avoid one or two word titles)

> search engines will display about 50-60 character of the title, so try not to have tiltes longer than that

> don not use just a list of words as the title

> try to make the title as accurate and meaningful as possible
