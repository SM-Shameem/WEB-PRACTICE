introduction

# the math object allows you to perform mathematical tasks 
# Math is not a constructor. all properties/methods of math can be called by using math as an object, 
  without creating it 
  like > var x = Math.PI; // returns PI
  like > var y = Math.sqrt(16); // returns the square root of 16 
--------------------------------------------------------------------------------
# there are eight (08) math object properties 
01. E                - Returns Euler's number (approx. 2.718)
02. LN2              - Returns the natural logarithm of 2 (approx. 0.693)
03. LN10             - Returns the natural logarithm of 10 (approx. 2.302)
04. LOG2E            - Returns the base-2 logarithm of E (approx. 1.442)
05. LOG10E           - Returns the base-10 logarithm of E (approx. 0.434)
06. PI               - Returns PI (approx. 3.14)
07. SQRT1_2          - Returns the square root of 1/2 (approx. 0.707)
08. SQRT2            - Returns the square root of 2 (approx. 1.414)
--------------------------------------------------------------------------------
# there are thirty-three (33) math object methods 
01. abs(x)               - Returns the absolute value of x
02. acos(x)              - Returns the arccosine of x, in radians
03. acosh(x)             - Returns the hyperbolic arccosine of x
04. asin(x)              - Returns the arcsine of x, in radians
05. asinh(x)             - Returns the hyperbolic arcsine of x
06. atan(x)              - Returns the arctangent of x as a numeric value between -PI/2 and PI/2 radians
07. atan2(y, x)          - Returns the arctangent of the quotient of its arguments
08. atanh(x)             - Returns the hyperbolic arctangent of x
09. cbrt(x)              - Returns the cubic root of x
10. ceil(x)              - Returns x, rounded upwards to the nearest integer
11. clz32(x)             - Returns the number of leading zeros in a 32-bit binary representation of x
12. cos(x)               - Returns the cosine of x (x is in radians)
13. cosh(x)              - Returns the hyperbolic cosine of x
14. exp(x)               - Returns the value of Ex
15. expm1(x)             - Returns the value of Ex minus 1
16. floor(x)             - Returns x, rounded downwards to the nearest integer
17. fround(x)            - Returns the nearest (32-bit single precision) float representation of a number
18. log(x)               - Returns the natural logarithmof x
19. log10(x)             - Returns the base-10 logarithm of x
20. log1p(x)             - Returns the natural logarithm of 1 + x
21. log2(x)              - Returns the base-2 logarithm of x
22. max(x, y, z, ..., n) - Returns the number with the highest value
23. min(x, y, z, ..., n) - Returns the number with the lowest value
24. pow(x, y)            - Returns the value of x to the power of y
25. random()             - Returns a random number between 0 and 1
26. round(x)             - Rounds x to the nearest integer
27. sign(x)              - Returns the sign of a number (checks whether it is positive, negative or zero)
28. sin(x)               - Returns the sine of x (x is in radians)
29. sinh(x)              - Returns the hyperbolic sine of x
30. sqrt(x)              - Returns the square root of x
31. tan(x)               - Returns the tangent of an angle
32. tanh(x)              - Returns the hyperbolic tangent of a number
33. trunc(x)             - Returns the integer part of a number (x)

# abs() - Math.abs(x) - ECMAScript1 
  parameter > (x) - required. a number 
  return    > a number, representing the absolute value of the specified number, 
              or NaN if the value is not a number, or 0 if the value is (null) 
  example   > Return the absolute value of a number:
  Math.abs(-7.25);
  example   > Return the absolute value of different numbers:
  var a = Math.abs(7.25);    // result 7.25
  var b = Math.abs(-7.25);   // result 7.25
  var c = Math.abs(null);    // result 0
  var d = Math.abs("Hello"); // result NaN
  var e = Math.abs(2+3);     // result 5
- The abs() method returns the absolute value of a number.

# acos() - Math.acos(x) - ECMAScript1
  parameter > x - required. a number 
  return    > a number, from 0 to PI, or NaN if the value is outside the range of (-1) to (1) 
  example   > Return the arccosine of a number:
  Math.acos(0.5);
- The acos() method returns the arccosine of a number as a value value between 0 and PI radians.
- Note: If the parameter x is outside the range -1 to 1, the method will return NaN.
- Tip: -1 will return the value of PI.

# acosh() - Math.acosh(x) - ECMAScript6 
  parameter > x - required, a nubmer 
  return    > a number, or NaN if the parameter is less than 1 
  example   > Return the hyperbolic arccosine of a number:
  Math.acosh(2);
- The acosh() method returns the hyperbolic arccosine of a number.
- Note: If the parameter x is less than 1, the method will return NaN.

# asin() - Math.asin(x) - ECMAScript1 
  parameter > x - required, a number 
  return    > a number, from (-PI/2) to (PI/2), or NaN if the value is outside the range of (-1) to (1) 
  example   > Return the arcsine of a number:
  Math.asin(0.5);
- The asin() method returns the arcsine of a number as a value between -PI/2 and PI/2 radians.
- Note: If the parameter x is outside the range -1 to 1, the browser will return NaN.
- Tip: 1 will return the value of PI/2. -1 will return the value of -PI/2.

# asinh() - Math.asinh(x) - ECMAScript6
  parameter > x - required, a number 
  return    > a number 
  example   > Return the hyperbolic arcsine of a number:
  Math.asinh(1);
- The asinh() method returns the hyperbolic arcsine of a number.

# atan() - Math.atan(x) - ECMAScript1 
  parameter > x - required. a number 
  return    > a number, from (-PI/2) to (PI/2), or NaN if the value is empty 
  example   > Return the arctangent of a specified number:
  Math.atan(2);
- The atan() method returns the arctangent of a number as a value between -PI/2 and PI/2 radians.

# atan2() - Math.atan2(y, x) - ECMAScript1
  parameter > y - required, a number representing the y coordinate 
  parameter > x - required, a number representing the x coordinate 
  return    > a number, from (PI) to (-PI), or NaN if the value(s) are empty 
  example   > Assume you had a point with the (x,y) coordinates of (4,8), you could calculate the angle in 
              radians between that point and the positive X axis as follows:
  Math.atan2(8, 4);
- atan2() returns the arctangent of the quotient of its arguments, as a numeric value between PI and -PI radians.
- The number returned represents the counterclockwise angle in radians (not degrees) 
  between the positive X axis and the point (x, y).
- Note: With atan2(), the y coordinate is passed as 1st argument and the x coordinate is passed as 2nd argument.

# atanh() - Math.atanh(x) - ECMAScript6
  parameter > x - required, a number 
  return    > a number, or NaN, or Infinity, or (-Infinity) 
  example   > Return the hyperbolic arctangent of a specified number:
  Math.atanh(0.5);
- The atanh() method returns the hyperbolic arctangent of a number.
- Note: If the parameter x is greater than 1, or less than -1, the method will return NaN.
- Note: If the parameter x is 1, the method will return Infinity.
- Note: If the parameter x is -1, the method will return -Infinity.

# cbrt() - Math.cbrt(x) - ECMAScript6

  example   > Return the cubic root of a number:
  Math.cbrt(125);
- The cbrt() method returns the cubic root of a number.

# ceil() - Math.ceil(x) - ECMAScript1 
  parameter > x - required, the number you want to round 
  return    > a number, representing the nearest integer when rounding upwards 
  example   > Round a number upward to its nearest integer:
  Math.ceil(1.4)
  example   > Use the ceil() method on different numbers:
  var a = Math.ceil(0.60);   // result 1 
  var b = Math.ceil(0.40);   // result 1 
  var c = Math.ceil(5);      // result 5 
  var d = Math.ceil(5.1);    // result 6
  var e = Math.ceil(-5.1);   // result -5
  var f = Math.ceil(-5.9);   // result -5
- The ceil() method rounds a number UPWARDS to the nearest integer, and returns the result.
- If the passed argument is an integer, the value will not be rounded.

# clz32() - Math.clz32(x) - ECMAScript 2015 
  parameter > x - required, a number 
  return    > The number of leading zeros bits in the 32-bit binary representation of a number. 
              If the number is 0, then this method returns 32 (as all bits are 0).
  example   > Return the number of leading zeros in a 32-bit binary representation of different numbers:
  var a = Math.clz32(0);
  var b = Math.clz32(1);
  var c = Math.clz32(2);
  var d = Math.clz32(4); 
- The clz32() method is short for CountLeadingZeroes32, 
  and it returns the number of leading zeros in a 32-bit binary representation of a number.

# exp() - Math.exp(x) - ECMAScript1
  parameter > x - required. a number

  var b = Math.exp(-1);
  var c = Math.exp(5);
  var d = Math.exp(10);
- exp() returns value of Ex, where E is Euler's number (approximately 2.7183) and x is the number passed to it.

# expm1() - Math.expm1(x) - ECMAScript 2015 
  parameter > x - required. a number 
  return    > a number, representing E(x) minus 1 
  example   > Find the exponential value of a number - 1:

  example   > Use the expm1() method on different numbers:

  var b = Math.exp(-1);
  var c = Math.exp(5);
  var d = Math.exp(10);
- The expm1() method returns the value of Ex minus 1, 
  where E is Euler's number (approximately 2.7183) and x is the number passed to it.
- This method is more accurate than using Math.exp() and subtracting 1.

# max() - Math.max(n1, n2, n3, ..., nX) - ECMAScript1
  parameter > n1, n2, ..., nX - optional. one or more numbers to compare 
  return    > a number, representing the highest number of the arguments, 
              or (-infinity) if no arguments are given, or NaN if one or more arguments are not numbers 
  example   > Return the number with the highest value:
  Math.max(5, 10);
  example   > Return the number with the highest value:
  var a = Math.max(5, 10);
  var b = Math.max(0, 150, 30, 20, 38);
  var c = Math.max(-5, 10);
  var d = Math.max(-5, -10);
  var e = Math.max(1.5, 2.5);
- The max() method returns the number with the highest value.
- Tip: The min() method returns the number with the lowest value.
  
# min() - Math.min(n1, n2, n3, ..., nX) - ECMAScript1
  parameter > n1, n2, n3, ..., nX - optional, one or more numbers to compare 
  return    > A Number, representing the lowest number of the arguments, 
              or Infinity if no arguments are given, or NaN if one or more arguments are not numbers
  example   > Return the number with the lowest value:
  Math.min(5, 10);
  example   > Return the number with the lowest value:
  var a = Math.min(5, 10);
  var b = Math.min(0, 150, 30, 20, 38);
  var c = Math.min(-5, 10);
  var d = Math.min(-5, -10);
  var e = Math.min(1.5, 2.5);
- The min() method returns the number with the lowest value.
- Tip: The max() method returns the number with the highest value.

# sign() - Math.sign(x) - ECMAScript 2015 
  parameter > x - required. a number 
  return    > a number, representing the sign of the specified number 
              - if the number is positive, it returns 1 
              - if the number is negative, it returns (-1)
              - if the number is positive zero, it returns 0 
              - if the number is negative zero, it returns (-0) 
              - if the number is not a number, it returns NaN
  example   > Find out if a number is negative or positive:
  var a = Math.sign(3);    // Returns 1 (positive)
  var b = Math.sign(-3);   // Returns -1 (negative)
  var c = Math.sign(0);    // Returns 0 (zero) 
- The sign() method checks whether a number is negative, positive or zero.
- If the number is positive, this method returns 1.
- f the number is negative, it returns -1.
- If the number is zero, it returns 0.
